# HTML Resume

A clean, responsive, and interactive resume webpage built with **HTML5**, **CSS3**, and **JavaScript**.  
This is my first project as I start my journey toward becoming a **Full Stack Developer**.

## 🚀 Live Demo
[View Resume](https://siddharthrao-dev.github.io/html-resume/)

---

## 📄 Features
- **Semantic HTML5** structure for accessibility and SEO
- **Responsive design** with CSS3 (mobile-friendly)
- **Interactive form** with JavaScript validation
- **Dynamic hover effects** for better user experience

---

## 🛠️ Technologies Used
- **HTML5** – Page structure
- **CSS3** – Styling and layout
- **JavaScript (ES6+)** – Form validation & interactivity
- **Git & GitHub** – Version control and hosting
